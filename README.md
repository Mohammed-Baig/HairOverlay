# HairOverlay

An application that allows the user to try on various hairstyles to see how they may look with one before getting a haircut. The application works by first getting a picture of your room, without you in it, and then filtering out the color blue. What this allows you to do is if you're wearing a blue hat that covers your hair, it allows you to completely
remove your current hairstyle from the equation and leave yourself a blank slater. At which point you can later choose what hairstyle you want to try on through user input. When
the hairstyle is chosen it will appear in the center of the screen and you can try it on.
